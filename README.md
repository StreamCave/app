# Comment installer le projet ?

## 1) Clone le projet GIT
``` 
git clone https://github.com/BrietGame/SA-Prod.git 
```
## 2) Installer toutes les dépendances JS
```
npm install -y
```
## 3) Installer toutes les dépendances PHP
```
composer install
```
